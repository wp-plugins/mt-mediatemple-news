== Description ==
Keep track of your (mt) Media Temple services, and get suggestions on how to optimize them for your WordPress site.

== Installation ==
1. Download and install the plugin from http://wordpress.org/plugins/mt-mediatemple-news/
1. Activate the plugin through the \"Plugins\" menu in WordPress.
1. Provide your (mt) Media Temple API via the settings page.
