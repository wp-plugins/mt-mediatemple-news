(mt) Media Temple is a web hosting provider, with this plugin you can stay up-to-date on the latest (mt) News.

== Description ==

This plugin currently offers several useful features. Four dashboard widgets: KnowledgeBase Search, Twitter Feed, System Status, and Latest News. Quick links to useful Media Temple tools, such as the AccountCenter or User Forums are added to the Wordpress Admin Menu.
